Constraint Filter — UI Approaches

Three ways to add per-zone include/exclude filtering to the Available Units panel. Each zone can be: neutral (no effect) · included (show only) · excluded (hide).

A

Toolbar Popover

A "GSP Filter" button sits in the existing toolbar. Clicking it opens a floating panel with all 14 zones — each has a 3-way toggle: neutral / include / exclude. The button shows a summary badge when active.

Pros

  • Zero permanent space — toolbar unchanged when inactive
  • Handles all 14 zones in scrollable list
  • Easy to extend with real constraint data later

Cons

  • Active filters hidden behind a click
  • Popover closes if you misclick
B

Collapsible Filter Strip

A second row appears below the toolbar. Collapsed by default — click "GSP" to expand. Each zone is a chip: click once for include (green), click again for exclude (red), again to reset. Active filters stay visible as chips even when collapsed.

C

Toolbar Button + Active Chips

A "GSP" button opens a checklist popover to select zones. Selected zones then appear as compact chips directly in the toolbar (with +/− toggle on each chip). The toolbar wraps naturally when filters are active.
