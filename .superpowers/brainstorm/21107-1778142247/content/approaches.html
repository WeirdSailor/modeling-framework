Pill row selects one reason code at a time. Summary bar above the table updates instantly — unit count, total MEL, estimated cost. Simple and fast to switch between reasons.

B — Summary cards + click-to-filter

All reason codes visible as cards at once — cost, unit count, total MW. Click any card to filter the table below. You see the full picture without switching tabs, and can drill in on one reason code.

C — Grouped table, no filter

Table is split into collapsible sections by reason code. Each section header shows the subtotal. No filtering — everything visible at once. Best if you regularly compare reasons side-by-side.

My recommendation: B — you see the full cost breakdown at a glance (which is the primary goal), and can still drill into any reason code without losing context of the others.
